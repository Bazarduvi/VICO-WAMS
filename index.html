📖 Manual de uso
🏠 INICIO — Obtener miniaturas
Pega la URL de un video de YouTube, Facebook, Instagram, TikTok, Vimeo o Twitter/X en el campo de texto y pulsa la lupa. Usa Pegar desde portapapeles para pegar automáticamente. Una vez obtenida la miniatura puedes compartirla en WhatsApp, descargarla o guardarla en la biblioteca.

💡 Truco para Instagram y Facebook: el botón 🖼️ Subir imagen / captura de pantalla es la opción más recomendada para estas redes. Haz un pantallazo del contenido que quieras compartir, luego pulsa ese botón, selecciona tu captura desde la galería y se abrirá el editor de recorte automático. Allí puedes recortar los bordes, ajustar la proporción y luego usar el editor para añadir texto, stickers y más antes de compartirlo.
✨ IA — Generar miniatura
Generar miniatura con IA: usa el título y descripción del video para crear una ilustración personalizada. Requiere API Key de Groq o Gemini (configúrala en Ajustes → IA).
Generar desde mi prompt: escribe una descripción libre y la IA genera la imagen.
Obtener imagen original con IA: analiza la URL, extrae el og:image si existe o genera una ilustración representativa del contenido.
✏️ Editor de miniaturas
Pulsa Añadir texto y stickers en el resultado para abrir el editor. Puedes: añadir texto con 15 fuentes y 5 estilos (CTA, Neón, Sombra, Contorno, Fuego), elegir color o gradiente, ajustar tamaño con el slider, mover elementos arrastrando o con pellizco para escalar, añadir stickers emoji y exportar la imagen editada.

Flujo recomendado para Instagram/Facebook: sube tu captura de pantalla con el botón 🖼️ Subir imagen → recórtala en el editor → pulsa "Recortar y usar" → pulsa "Añadir texto y stickers" para personalizarla → comparte.
📚 Biblioteca
Guarda miniaturas con el botón Biblioteca. Asigna categorías para organizarlas. Usa el buscador para filtrar. Desde cada tarjeta puedes compartir, descargar o eliminar.
⚙️ Ajustes
Tema visual: TRON (cian), MATRIX (verde), GOLD (dorado).
Personalizar botones: forma (esquinas) y color primario.
Preferencias: sonido, efecto fondo, brillo, calidad miniatura, auto-pegar, historial, auto-guardar, abrir video al tocar.
IA: configura tu proveedor (Groq/Gemini), modelo y API Key. Personaliza el prompt plantilla.
Datos: exporta/importa tu biblioteca en JSON.
Idioma: cambia el idioma de toda la interfaz.
📲 Instalar como app
WAMS es una PWA. En Chrome/Edge pulsa el botón 📲 en la cabecera o el banner de instalación para instalarla en tu dispositivo como app nativa.
